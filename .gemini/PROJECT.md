# LegalFlow - Visão do Produto e Contexto

## O que é
LegalFlow é um SaaS (Software as a Service) de gestão jurídica focado em advogados autônomos e pequenos a médios escritórios. 

## Diferencial Competitivo (A Filosofia)
Ao contrário dos ERPs jurídicos tradicionais que funcionam apenas como "bancos de dados burocráticos", o LegalFlow atua como um Sistema Operacional de Fluxo e Retenção. O foco não é apenas ajudar o advogado a escrever peças, mas sim resolver os gargalos reais do dia a dia.

## Pilares Principais (Core Features)
1. **Intake Zero-Click (Cadastro sem Atrito):** - Criação de clientes e processos na mesma tela.
   - Integração com a API pública do CNJ para autopreenchimento de dados processuais (Tribunal, Comarca, Assuntos, Movimentações).
2. **Legal AI para Client Success:** - Uso de IA não apenas para pesquisa jurídica estruturada (Base Legal), mas para traduzir o "juridiquês" das movimentações em resumos simples e executivos para envio aos clientes via WhatsApp.
3. **Financeiro com Split Inteligente:**
   - Resolução da confusão patrimonial típica de escritórios.
   - Cálculo automático da % de honorários (quota litis), separando o que é Receita do Escritório (Honorários) do que é Passivo/Conta a Pagar (Dinheiro de Repasse do Cliente).
4. **Arquivamento e Métricas:**
   - Soft delete de processos ("Arquivamento") com exigência de registro do resultado (Procedente, Improcedente, Acordo) para manter o histórico e gerar métricas de taxa de sucesso.
